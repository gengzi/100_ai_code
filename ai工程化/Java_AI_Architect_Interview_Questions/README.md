## 💻 代码使用指南

### 环境要求

- **Java**: JDK 17+ (推荐使用OpenJDK或Oracle JDK)
- **构建工具**: Maven 3.8+ 或 Gradle 7+
- **IDE**: IntelliJ IDEA或Eclipse
- **AI框架**: 根据具体项目需求安装相应的依赖

### 运行示例代码

```bash
# 克隆项目
git clone https://github.com/your-repo/Java_AI_Architect_Interview_Questions.git
cd Java_AI_Architect_Interview_Questions

# 选择一个模块进行学习

javac -cp ".:lib/*" *.java
java -cp ".:lib/*" MainClass
```
