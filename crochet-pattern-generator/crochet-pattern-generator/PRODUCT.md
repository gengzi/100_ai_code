# 钩针图解生成器 - 产品文档

## 📋 产品概述

### 产品名称
钩针图解生成器 (Crochet Pattern Generator)

### 版本信息
- 当前版本：1.0.0
- 发布状态：开发中

### 产品描述
一个创新的Web应用程序，可以将用户上传的图片自动转换为钩针编织图解。该工具为手工爱好者和专业编织师提供便捷的图案设计解决方案。

## 🎯 核心功能

### 主要特性
1. **图片转换**：支持将各种格式的图片转换为钩针编织图解
2. **自定义尺寸**：允许用户设置编织图案的尺寸规格
3. **颜色选择**：提供颜色定制功能，满足不同编织需求
4. **导出功能**：支持多种格式的图案导出

### 技术栈
- **前端框架**：React 18.2.0
- **构建工具**：Vite 4.4.5
- **样式框架**：Tailwind CSS 3.3.3
- **类型支持**：TypeScript 5.0.2
- **图像处理**：html2canvas
- **PDF生成**：jsPDF
- **文件上传**：react-dropzone
- **UI组件**：Headless UI + Lucide React
